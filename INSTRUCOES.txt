## APÓS COPIAR OS ARQUIVOS

No CMD, dentro da pasta do projeto:

  flutter clean
  flutter pub get
  flutter build appbundle --release

O AAB estará em:
  build/app/outputs/bundle/release/app-release.aab

## O QUE AINDA FALTA FAZER MANUALMENTE (fora dos arquivos)

1. Firebase Console → Authentication → SHA-1 do keystore de release
   Comando para pegar o SHA-1:
   keytool -list -v -keystore android/app/conectapro.jks -alias conectapro

2. google-services.json já tem com.conectapro.app cadastrado — OK

3. Play Console:
   - Criar conta de desenvolvedor (se ainda não tiver) — taxa única de U$25
   - Criar novo app → "ConectaPro"
   - Preencher ficha do app: descrição, categoria (Negócios ou Estilo de vida), ícone 512x512, screenshots
   - Política de privacidade — já tem (politica_privacidade_page.dart), hospedar em URL pública
   - Declarar permissões: POST_NOTIFICATIONS → explicar por quê
   - Subir o AAB em "Versão interna" para teste primeiro

4. Versão do app no Firestore (auto-update):
   Atualizar collection config/app → { versao: "1.0.1", apkUrl: "..." }
